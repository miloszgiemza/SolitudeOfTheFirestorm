Readme EN
Main repository folder contains folder  "Project" with Unity LTS 2022.3.20f1 project of the game and Android installation file "Solitude.apk" with some test levels presets.   
"Solitude" is a prototype in developement.

Game core principles:
-Tactical, turn-based gameplay on grid
-"Reversed Tetris" feeling - hordes of enemies are coming from the top of the screen and we need to get rid of them as fast as possible with area spells.
-Tower defence game elements (more may be added yet).
-We play as an eccentric wizard whose lonely tower happens to be in the way of the army of darkness. Out of my lawn!

Implemented mechanics:
-Working gameplay loop, progression and save system.
-Universal input handling for mobile amnd desktop devices based on new Unity Input System package.
-Scriptable Object and Unity Editor based tools for game and level designers for creating content without the need to code.
-System of creating rigidly set or partially randomised levels based on Scriptable Objects.  
-Spells (offensive, creating obstacles and debuffing enemies).
-Enemies pathfinding use graph search algorithms. 
-Enemies and player attributes.
-Player inventory and three types of items - big scrolls, mixtures and small scrolls. Some can be used instead of main action (big scrolls) and some as a secondary action.
-Enemies and player can receive statuses. 
-Enemies have different stats, bahaviours and can buff each other.
-Loot system.
 
Redame PL
Główny folder repozytorium zawiera folder "Project" z projektem Unity LTS 2022.3.20f1 gry oraz plik instalacyjny aktualnej wersji gry dla systemu Android z kilkoma testowymi poziomami. 
"Solitude" jest prototypem gry w trakcie produkcji.

Założenia gry:
-Taktyczna, turowa rozgrywka na siatce (kafelki).
-Założenie "odwróconego Tetrisa" - hordy wrogów nadciągają z góry mapy, a my czyścimy je zaklęciami obszarowymi o konkretnych kształtach. 
-Rozgrywka ma elementy "Tower-Defence" (których w przyszłości może być dodanych więcej). 
-Wcielamy się w ekscentrycznego czarodzieja, którego wieża znalazła się na drodze armii ciemności.

Zaimplementowane mechaniki:
-Działający gameplay loop, progresja rozgrywki i system zapisu progresji. 
-Bazujący na Scriptable Objectach i Edytorze Unity system do pracy dla game i level designerów niewymagjący programowania. 
-Możliwość tworzenia "sztywnych" i randomizowanych presetów poziomów przy użyciu Scriptable Objectów.
-Uniwersalne sterowanie dla urządzeń mobilnych i desktopowych bazujące na nowej paczce Unity Input System.
-Zaklęcia (ofensywne, tworzące przeszkody i nakładające statusy na wrogów). 
-Zachowania wrogów bazują na algorytmach wyszukiwania ścieżek na grafach. 
-System atrybutów gracza i wrogów. 
-System ekwipunku gracza i system przedmiotów. Trzy główne typy przedmiotów: Duże Zwoje, Mikstury i Małe Zwoje. Niektóre przedmioty mogą być użyte zamiast akcji głównej w turze (Duże Zwoje), a inne w ramach akcji dodatkowej.
-System nakładania statusów na wrogów i gracza.
-Wrogowie mają różne zachowania, statystyki i mogą buffować się nawzajem.
-System lootu.
